신입사원 업무용 PC 프로그램 설치 및 세팅 가이드
본 가이드는 증권업무 수행에 필수적인 소프트웨어의 설치 순서와 세팅 방법을 안내합니다. 모든 설치는 회사의 보안 정책(DRM, DLP 등) 내에서 수행되어야 합니다.
1. 필수 업무용 프로그램 리스트

| 분류 | 프로그램명 | 주요 용도 |
| --- | --- | --- |
| 트레이딩/시장분석 | HTS (Home Trading System) | 실시간 주가 확인, 호가 분석, 종목 뉴스 모니터링 |
| 금융 정보 단말기 | 연합인포맥스 / 블룸버그 | 거시 경제 데이터, 채권 금리, 해외 시장 실시간 지표 분석 |
| 협업 및 소통 | 사내 메신저 / MS Teams | 부서 간 소통 및 타 증권사(브로커)와의 업무 연락 |
| 문서/데이터 | MS Office (Excel 위주) | 모델링, 리포트 작성, 대량의 시세 데이터 가공 |
| 정보 보안 | DRM / DLP / 백신 | 사외 문서 유출 방지 및 내부 정보 보호(자동 설치 확인 필수) |

2. 프로그램별 세팅 단계 (Step-by-Step)
[Step 1] HTS (홈트레이딩 시스템) 설치 및 환경 설정
설치: 키움증권 인트라넷 또는 공식 홈페이지에서 전문가용 HTS를 다운로드하여 설치합니다.
멀티 모니터 세팅: 증권사 업무는 보통 3~4대의 모니터를 사용합니다. [화면 레이아웃] 설정에서 주요 지수, 관심 종목 호가창, 뉴스 창을 각각 배치한 후 **'레이아웃 저장'**을 반드시 수행하세요.
폰트 및 색상: 가독성을 위해 호가창 폰트를 조정하고, 상승(빨강)/하락(파랑) 색상을 한국 표준에 맞게 확인합니다.
[Step 2] 정보 단말기 (인포맥스/블룸버그) 로그인
ID 신청: 팀장 승인 후 IT 지원 부서로부터 받은 고유 ID로 로그인합니다.
엑셀 Add-in 연결: 엑셀 상단 탭에 '인포맥스' 혹은 'Bloomberg' 탭이 생성되었는지 확인합니다. 실시간 데이터를 엑셀로 불러오기 위한 필수 과정입니다.
[Step 3] 엑셀(Excel) 최적화 세팅
파일 > 옵션 > 수식: '반복 계산 사용' 체크 (순환 참조 계산이 필요한 금융 모델링 시 필요).
빠른 실행 도구 모음: [값 붙여넣기], [서식 복사], [틀 고정] 등 자주 쓰는 기능을 상단에 배치합니다.
[Step 4] 보안 프로그램 확인
문서 암호화(DRM): 엑셀이나 워드 파일을 저장할 때 자동으로 회사 마크가 붙으며 암호화되는지 확인하세요. (외부로 파일을 보낼 때는 반드시 별도의 해제 승인 절차를 거쳐야 합니다.)
3. 신입사원 세팅 체크리스트 (Final Check)
[ ] 키움증권 인트라넷 및 그룹웨어 로그인 정상 여부
[ ] 부서 공용 폴더(NAS/Shared Drive) 접근 권한 확인
[ ] 법인 이메일 서명 설정 (성명, 부서, 직통 번호, 회사 로고 포함)
[ ] 복합기(프린터) 드라이버 연결 및 스캔 기능 테스트
